obs-package-update is a simple python library for updating packages in the Open Build Service by wrapping the osc command line client.

This package is intended for easing updating packages automatically from python without having to mess with osc itself.

UsageÂ¶

The core functionality is provided by the Updater class, where you have to implement the add_files() function. This asynchronous function obtains the destination to the checked out package and should write the updated sources into that directory. To then perform the actual update, execute update_package().

A very simple (and rather dumb) example for such an updater would be the following:

class Eraser(Updater):

    async def add_files(self, destination: str) -> List[str]:
        await run_cmd(f"rm -rf {destination}")
        return []
